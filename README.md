## 📖 About

This repository contains examples demonstrating **Hibernate** and **JDBC** for Java database applications.

### ✨ Features

- JDBC Connectivity
- Hibernate ORM
- CRUD Operations
- Entity Mapping
- MySQL Integration
- Maven Project

---

## 🛠️ Tech Stack

- Java
- JDBC
- Hibernate
- MySQL
- Maven

---

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/anandsagar6/Hibernate-and-JDBC.git
```

Build the project:

```bash
mvn clean install
```

Run the application from your IDE.
