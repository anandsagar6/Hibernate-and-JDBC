# 📖 About

This repository is a **hands-on learning resource** for mastering **JDBC** and **Hibernate ORM** in Java.

It demonstrates how Java applications communicate with relational databases using both:

- 🔹 **JDBC (Java Database Connectivity)** – Direct SQL-based database access.
- 🔹 **Hibernate ORM** – Object Relational Mapping for simplified and efficient database operations.

Whether you're a beginner learning Java databases or preparing for interviews, this repository provides practical examples and best practices.

# ⚖️ JDBC vs Hibernate

| Feature | JDBC | Hibernate |
|---------|------|-----------|
| Database Access | SQL Queries | ORM Framework |
| Learning Curve | Easy | Moderate |
| SQL Writing | Manual | Automatic |
| Boilerplate Code | High | Low |
| Performance | Faster for Simple Queries | Better for Large Applications |
| Portability | Database Specific | Database Independent |
| Productivity | Lower | Higher |
